## Теорема (арифм. свойства непрерывных функций)
Пусть $f,g:~D\to\mathbb{R}~~f,g:~в~x_0\in D$ =>
1) $f + g;~f - g;~fg;~|f|$ - непрерывны в $x_0$
2) $g(x_0)\neq0$ => $\frac{f}{g}$ - непрерывна в $x_0$

Доказательство:

*Случай 1*:
$x_0$ - изолированная точка $D$ => любая функция, определенная на $D$, непрерывна в $x_0$, в частности $f+g;~f-g;~fg;~|f|;~\frac{f}{g}$

*Случай 2*:
Пусть $x_0\in D$ - предельная точка $D$ => утверждение следует из арифм. свойств предела, например $\frac{f(x_0)}{g(x_0)}=\frac{\lim_{x\to x_0}f(x)}{\lim_{x\to x_0}g(x)} = \lim_{x\to x_0}\frac{f(x)}{g(x)}$, т.к. $g(x_0)\neq0$.
